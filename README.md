## GK File Utils

Short library for file & images manipulation.

Language: Java

## Features

- [x] Read file content
- [x] Write file content
- [x] Read image content
- [x] Resizing images 
- [x] Change images format

## Maven 

```
<dependency>
    <groupId>gk.os</groupId>
    <artifactId>files</artifactId>
    <version>2.0.0</version>
</dependency>
```
